# Padrões de Estilo do Projeto

Esta pasta contém os arquivos de estilo e configurações visuais do app, conforme o padrão definido no PROJETO.MD.

- `AppStyles.ts`: Exporta constantes de cores, tipografia, espaçamento e componentes para uso global.
- `TailwindConfig.js`: Referência de classes Tailwind para uso em React Native.

Utilize esses arquivos para garantir consistência visual e facilitar a manutenção do layout.
