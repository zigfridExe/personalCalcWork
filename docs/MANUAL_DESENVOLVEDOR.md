# Manual do Desenvolvedor

Este manual explica a arquitetura, padrões e boas práticas do projeto.

## Arquitetura
- O projeto segue o padrão DDD (Domain-Driven Design).
- O código está organizado em domínios dentro de `src/domains`.
- Recursos compartilhados ficam em `src/shared`.
